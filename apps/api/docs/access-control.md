# 권한 정책

## 권한 종류

| role | 설명 |
|---|---|
| `USER` | 일반 사용자 |
| `ADMIN` | 관리자 |
| `SUPER_ADMIN` | 최고 관리자 |

## 관리자 권한

관리자는 다음 기능을 사용할 수 있다.

- 게시글 신고 처리
- Q&A 답변
- 위키 관리

## 최고 관리자 권한

최고 관리자는 다음 기능만 사용할 수 있다.

- 관리자 임명
- 배너 관리

## 최고 관리자 권한 부여

최고 관리자 권한은 관리자 화면이나 API가 아니라 DB에서 수동으로 업데이트한다.

비밀번호 같은 민감 정보는 문서, 소스 코드, 마이그레이션 파일에 기록하지 않는다.
